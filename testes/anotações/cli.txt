CLI

O que é?

Command Line Interface ( Interface de linha de comando).

Forma de comunicar diretamente com o sistema, servidor, através de uma linha de comando realizado por texto.

Usado para criar pastas, navegar entre elas e fazer varias manipulações. Visualizar banco de dados.
